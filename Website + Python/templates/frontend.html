MQTT Testing Kit

Welcome

This website is an online test-kit designed to help understand how vulnerable MQTT can be, and to educate users on how to prevent security issues.
There are three sections of the web-page to explore:



-The educational sector, this will provide valuable information on MQTT, what it is, and how to secure networks using it.


-The testing sector, this page contains an embedded test kit, that will allow a user to input an IP twice, once to test if a MQTT device or network
is susepctable to attacks (open port), the other to bounce off Shodan's API to see if the device or network is listed, and/or considered insecure by the search engine.


-The about sector, this page will give a brief overview of the page intentions, the author and the test kit itself.

This website has been desinged to be inclusive to all users, however does not currently work on mobile.
Resizing the page may shrink elements, but core functionaility will reside.

How This Test Kit Can Help You

This test kit aims to help both small scale and large scale clients, it is completely free to use.

Individuals may want to use this test kit to see whether their locally hosted servers or personal devices are visible
to others, and then further more whether shodan has their devices tagged as vulnerable on its database.

Organisations may want to use this test kit primarily for the second function, as shodan can quite commonly be used by hackers to find vulnerable
targets.

More info on shodan in the about page.
